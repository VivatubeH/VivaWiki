# 프로젝트1 : 게시판 

## 기술 스택
- Spring Boot 3.x
- Spring Security
- JWT
- JPA
- QueryDSL
- MySQL
- Swagger

## 프로젝트 기반 설정
- Spring Boot 프로젝트 구성 (의존성 세팅)
- DB 연결 및 JPA 설정
- 공통 응답 형식 구성 ApiResponse<T>
- 전역 예외 처리 @ControllerAdvice + 커스텀 Exception 클래스
- Swagger (SpringDoc OpenAPI) 설정
- CORS 설정
- JPA Auditing 설정 (createdAt, updatedAt 자동화)

## 회원 기능
- 회원가입 (이메일 + 비밀번호, BCrypt 암호화)
- 이메일 중복 검사 API
- 로그인 → JWT Access Token + Refresh Token 발급
- Access Token 재발급 (Refresh Token 검증)
- 로그아웃 (Refresh Token 무효화)
- 내 정보 조회
- 닉네임 / 비밀번호 수정
- 회원 탈퇴

## 카테고리 기능
- 카테고리 생성 / 수정 / 삭제 (관리자 전용)
- 카테고리 목록 조회
  
## 게시글 기능
- 게시글 작성 (카테고리 지정)
- 게시글 수정 (작성자 본인만)
- 게시글 삭제 (작성자 본인 + 관리자)
- 게시글 단건 조회 (조회수 증가)
- 게시글 목록 조회 (페이지네이션, 최신순/조회수순 정렬)
- 게시글 검색 (제목 / 내용 / 작성자 닉네임)
- 카테고리별 게시글 조회

## 댓글 기능
- 댓글 작성
- 댓글 수정 (작성자 본인만)
- 댓글 삭제 (작성자 본인 + 관리자)
- 게시글별 댓글 목록 조회
- 대댓글 작성 / 조회 (1 depth, parent_id 방식)

## 파일 첨부 기능
- 이미지 파일 업로드 (로컬 or S3)
- 게시글 삭제 시 첨부파일 연동 삭제
- 파일 다운로드

## 관리자 기능
- 회원 목록 조회
- 회원 강제 탈퇴
- 게시글 강제 삭제
- 댓글 강제 삭제

## 기술적 구현 포인트 (면접 어필용)
- JWT Access / Refresh Token 이중 구조
- Spring Security Filter Chain 커스텀 설정
- AuthenticationEntryPoint, AccessDeniedHandler 직접 구현 (401/403 분리)
- @Valid + 커스텀 Validator로 입력값 검증
- QueryDSL로 검색 동적 쿼리 처리
- Service 레이어 단위 테스트 (JUnit5 + Mockito)
