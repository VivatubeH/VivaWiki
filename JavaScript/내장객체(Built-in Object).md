내장객체(Built-in Object)
----------------------------------
- 브라우저의 자바스크립트 엔진에 기본적으로 내장된 객체륾 말합니다.
- 내장 객체의 종류 : 문자(String), 날짜(Date), 배열(Array), 정규 표현 객체(RegExp Object)

내장객체를 지원하는 이유
----------------------------------
자바스크립트에서의 필수적이고 기본적인 작업을 수행할 때 사용됩니다. 

내장객체의 생성법
---------------------------------
```javascript
참조변수 = new 생성 함수();
```
