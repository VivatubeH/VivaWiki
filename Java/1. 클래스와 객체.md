## 1. 클래스와 객체 - 핵심 개념

### 왜 클래스가 필요한가?

**문제 상황:**
```java
// 학생 정보를 변수로 관리 - 나쁜 예시
String student1Name = "학생1";
int student1Age = 15;
int student1Grade = 90;

String student2Name = "학생2";
int student2Age = 16;
int student2Grade = 80;
```

**문제점:**
- 학생이 추가될 때마다 변수 3개씩 계속 선언해야 함
- 데이터가 흩어져 있어 관리가 어려움
- 사람이 이해하기 어려운 구조

---

### 클래스로 해결
```java
public class Student {
    String name;
    int age;
    int grade;
}
```

**사용 예시:**
```java
Student student1 = new Student();
student1.name = "학생1";
student1.age = 15;
student1.grade = 90;
```

---

## 2. 핵심 용어 정리

### 클래스 (Class)
- **설계도** 또는 **틀**
- 객체를 만들기 위한 템플릿
- 예: `Student` 클래스

### 객체 (Object)
- 클래스를 기반으로 **실제로 만들어진 것**
- 메모리에 실제로 존재하는 실체
- 예: `student1`, `student2`

### 인스턴스 (Instance)
- 객체와 같은 의미지만, **특정 클래스로부터 생성되었음을 강조**할 때 사용
- "student1은 Student의 인스턴스다"

### 멤버 변수 (Member Variable) = 필드 (Field)
- 클래스에 속한 변수
- 예: `name`, `age`, `grade`

---

## 3. 객체 생성과 사용

### 객체 생성 3단계
```java
// 1. 변수 선언
Student student1;

// 2. 객체 생성 (new 키워드)
student1 = new Student();

// 3. 참조값 보관
// student1 변수에 객체의 메모리 주소(참조값)가 저장됨
```

**한 줄로 작성:**
```java
Student student1 = new Student();
```

---

### 참조값 (Reference)
```java
Student student1 = new Student(); // x001 이라는 참조값 저장됨
```

**핵심:**
- 변수에는 **객체 자체**가 아니라 **객체의 위치(참조값)**가 저장됨
- `student1 = x001` (실제로는 16진수 메모리 주소)

**참조값 확인:**
```java
System.out.println(student1);
// 출력: class1.Student@7a81197d
//      패키지.클래스@참조값(16진수)
```

---

### 객체 사용 - dot(.) 연산자
```java
// 값 대입
student1.name = "학생1";
student1.age = 15;
student1.grade = 90;

// 값 읽기
System.out.println("이름: " + student1.name);
System.out.println("나이: " + student1.age);
```

**동작 원리:**
```java
student1.name = "학생1";
↓
x001.name = "학생1";  // 참조값을 통해 실제 객체에 접근
```

## 4. 배열과 객체 함께 사용하기

### 객체 배열 생성
```java
// 1. 배열 선언
Student[] students = new Student[2];

// 2. 객체 생성 및 배열에 저장
students[0] = new Student();
students[0].name = "학생1";
students[0].age = 15;

students[1] = new Student();
students[1].name = "학생2";
students[1].age = 16;
```

---

### 배열 선언 최적화
```java
// 방법 1: 기본
Student[] students = new Student[]{student1, student2};

// 방법 2: 간결하게 (선언과 동시에 초기화할 때만 가능)
Student[] students = {student1, student2};
```

---

### for문으로 반복 처리

**일반 for문:**
```java
for (int i = 0; i < students.length; i++) {
    System.out.println("이름: " + students[i].name);
}
```

**향상된 for문 (Enhanced for loop) - 추천:**
```java
for (Student s : students) {
    System.out.println("이름: " + s.name + " 나이: " + s.age);
}
```

## 5. 면접 예상 질문

### Q1. 클래스와 객체의 차이는?
**A:** 
- **클래스**는 객체를 만들기 위한 설계도(틀)입니다.
- **객체**는 클래스를 기반으로 실제 메모리에 생성된 실체입니다.
- 예: `Student` 클래스(설계도) → `student1` 객체(실제 학생)

### Q2. 참조형 변수에는 무엇이 저장되나요?
**A:** 
- 객체 자체가 아니라 **객체의 메모리 주소(참조값)**가 저장됩니다.
- `Student student1 = new Student();`에서 `student1`은 객체의 위치를 가리키는 참조값을 가집니다.

### Q3. new 키워드의 역할은?
**A:** 
- 클래스를 기반으로 **힙(Heap) 메모리에 실제 객체(인스턴스)를 생성**합니다.
- 생성된 객체의 **참조값(메모리 주소)을 반환**합니다.

## 6. 실무 핵심 포인트

### ✅ 클래스 명명 규칙
```java
// Good - 대문자 시작, CamelCase
public class Student { }
public class MemberService { }

// Bad
public class student { }
public class member_service { }
```

### ✅ 변수는 참조값만 복사된다
```java
Student s1 = new Student();
s1.name = "학생1";

Student s2 = s1;  // 참조값만 복사됨!
s2.name = "학생2";

System.out.println(s1.name);  // "학생2" 출력 (같은 객체를 가리킴)
```

### ✅ null - 참조값이 없음
```java
Student student = null;  // 아직 객체를 가리키지 않음
System.out.println(student.name);  // NullPointerException 발생!
```

## 7. 실전 예제 코드
```java
// Student.java
public class Student {
    String name;
    int age;
    int grade;
}

// Main.java
public class Main {
    public static void main(String[] args) {
        // 객체 배열 생성
        Student[] students = new Student[3];
        
        // 객체 생성 및 초기화
        students[0] = new Student();
        students[0].name = "김철수";
        students[0].age = 20;
        students[0].grade = 90;
        
        students[1] = new Student();
        students[1].name = "이영희";
        students[1].age = 21;
        students[1].grade = 85;
        
        students[2] = new Student();
        students[2].name = "박민수";
        students[2].age = 22;
        students[2].grade = 95;
        
        // 향상된 for문으로 출력
        for (Student s : students) {
            System.out.println("이름: " + s.name + 
                             ", 나이: " + s.age + 
                             ", 성적: " + s.grade);
        }
    }
}
```
