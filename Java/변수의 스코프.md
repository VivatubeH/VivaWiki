변수의 스코프
===========

#### 스코프(Scope)
- 변수의 생존 가능한 범위(접근 범위)를 말합니다.

#### 스코프를 사용하는 이유
- 목적에 따라 사용한 변수는 더 이상 필요 없기 때문에 메모리에서 제거할 필요가 있습니다. ( 메모리 효율성 )

#### 형변환
- 자바에서 작은 범위에서 큰 범위로의 값 대입은 묵시적으로 이루어집니다. ( 형 변환을 명시적으로 사용할 필요는 없습니다. )
- 이는 작은 범위에서 큰 범위로의 대입은 기존의 데이터의 손실이 발생하지 않기 때문입니다.
- 반면 큰 범위에서 작은 범위로의 값 대입은 명시적으로 이루어져야 합니다. ( 값 손실이 발생하는 걸 인지하고 있음을 알려야 합니다. )

#### 자바에서의 연산 규칙
- 같은 타입끼리의 연산은 같은 타입의 결과가 나옵니다.
- 서로 다른 타입끼리의 연산은 더 큰 범위로 자동 형변환이 이루어지게 되어있습니다.
