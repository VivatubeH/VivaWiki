## String 클래스
- 자바에서 문자열을 처리하기 위해 사용하는 클래스
- java.lang 패키지로 제공되기 때문에, 별도의 import 없이 사용 가능함.

## String 클래스 사용 시 주의점

- String은 불변 클래스이므로 부분적인 변경이 불가능하고, 새로운 문자열이 생성되는 특징이 있다.

- new로 생성 시에는 Heap에 새로운 문자열 객체가 생성되고, 리터럴로 생성 시에는 constant String pool에 있을 시 중복 생성하지 않는다.
- 그로 인해, 주소 비교 시 상이해질 수 있다.

## String 클래스 사용법.

```java
import java.io.*;
import java.util.*;

public class Main {
    public static void main(String[] args) {
        String str = "Hello Java, String Class";

        // 1. index번째 문자 반환 -> charAt(int index)
        char ch = str.charAt(4);
        System.out.println(ch);

        // 2. 문자열의 길이 -> length()
        int length = str.length();
        System.out.println(length);

        // 3. 문자열의 동등성 비교 -> equals(Object anObject)
        boolean isEqual = str.equals("Hello Java, String Class");
        System.out.println(isEqual);

        // 4. 문자열의 사전순 비교 -> compareTo(String anotherString)
        int result1 = str.compareTo("A"); // 사전에서 뒤에 나오면 양수
        int result2 = str.compareTo("Z"); // 사전에서 앞에 나오면 음수
        int result3 = str.compareTo("Hello Java, String Class"); // 사전에서 같은 위치면 0
        System.out.println(result1);
        System.out.println(result2);
        System.out.println(result3);

        // 5. 문자열을 char 배열로 -> toCharArray()
        char[] charArray = str.toCharArray();
        System.out.println(charArray[2]);

        // 6. 모든 문자열을 소문자로 변경한 새 문자열 -> toLowerCase()
        String lowerString = str.toLowerCase();
        System.out.println(lowerString);

        // 7. 모든 문자열을 대문자로 변경한 새 문장려 -> toUpperCase()
        String upperString = str.toUpperCase();
        System.out.println(upperString);

        // 8. 해당 문자열의 포함 여부 -> contains()
        boolean isContain = str.contains("Hello");
        System.out.println(isContain);

        // 9. 문자열의 target을 변경 -> replace(CharSequence target, CharSequence replacement)
        String replacedString = str.replace("java", "python");
        System.out.println(replacedString);

        // 10. 문자열의 일부 추출 -> substring(int beginIndex, int endIndex) / 범위 [beginIndex, endIndex)
        String substring = str.substring(0, 4);
        System.out.println(substring);

        // 11. fromIndex부터 ch 문자의 첫 등장 인덱스 -> indexOf(int ch, int fromIndex)
        int firstIndex = str.indexOf('l', 0);
        System.out.println(firstIndex);

        // 12. 특정 구분자로 잘라서 배열로 만들기 -> split("구분자")
        str = "apple banana orange";
        String[] arr = str.split(" ");
        System.out.println(arr[0] + " " + arr[1] + " " + arr[2]);

        // 13. 접두사/접미사 체크하기 -> startsWith(), endsWith()
        String s = "Hello.java";
        System.out.println(s.startsWith("Hello"));
        System.out.println(s.endsWith(".java"));

        // 14. 문자열 앞뒤 공백 제거 -> trim(), strip()
        s = "    Hello    ";
        System.out.println(s);
        System.out.println(s.trim());
        System.out.println(s.strip()); // strip은 유니코드 공백까지 제거 ( Java 11 도입 )

        // 15. 문자열 반복 -> repeat(int count)
        s = "abc";
        System.out.println(s.repeat(3)); // repeat도 자바11 도입

        // 16. 배열을 문자열로 -> String.join("구분자", 배열)
        String[] arr2 = {"A", "B", "C"};
        System.out.println(String.join("-", arr2));

        // 17. 길이가 0인지, 공백인지 확인 -> isEmpty(), isBlank()
        String str1 = "";
        String str2 = " ";
        System.out.println(str1.isEmpty());
        System.out.println(str2.isEmpty()); // isEmpty는 길이가 0인지 체크함. ""만 true
        System.out.println(str1.isBlank());
        System.out.println(str2.isBlank()); // isBlank는 공백만 있는지 체크함. " "도 true, 자바 11 도입

        // 18. 정규표현식 사용 -> replaceAll(String regex, String replacement)
        str = "a1b2c3";
        System.out.println(str.replaceAll("[0-9]", "")); // 숫자만 제거
    }
}
```

## StringBuilder 사용법 

- 문자열을 사용할 때, + 연산자를 통한 문자열 결합은 매번 새로운 객체를 생성하므로 시간 초과의 확률이 높아짐.
- 그래서 StringBuilder의 사용법을 알면 좋다.

```java
import java.io.*;
import java.util.*;

public class Main {
    public static void main(String[] args) {
        StringBuilder original = new StringBuilder("apple,banana,orange");

        // 1. 문자열을 뒤에 추가하기. -> append(String str)
        original.append(",pineapple");
        System.out.println(original);

        // 2. 특정 위치의 문자열을 삭제하기. -> delete(start, end)
        original.delete(19, 29); // 끝 인덱스 미포함
        System.out.println(original);

        // 3. 특정 위치에 문자열 끼워넣기 -> insert(인덱스, 문자열)
        original.insert(6, "pineapple,"); // index의 앞으로 삽입
        System.out.println(original);

        // 4. 특정 위치의 문자열을 변경하기 -> replace(시작, 끝, 문자열)
        original.replace(3, 4, "st");
        System.out.println(original);

        // 5. 특정 위치의 문자를 변경하기
        original.deleteCharAt(3); // 3번째 문자 날리기
        original.setCharAt(3, 'l'); // 변경된 인덱스 반영해서 문자 변경
        System.out.println(original);

        // 6. 문자열 뒤집기 -> reverse()
        original.reverse();
        System.out.println(original);

        // 7. StringBuilder를 String으로 변환 -> toString()
        String str = original.toString();
        System.out.println(str);

        // 8. StringBuilder의 초기화 -> setLength(0)
        // -> StringBuilder 객체 추가 생성 없이, 내용만 비운다. ( 메모리 및 속도 효율 )
        original.setLength(0);
        System.out.println(original);
    }
}
```

## StringBuilder 사용 시 주의점
- insert, delete 등 사용 시 변화되는 인덱스를 생각해서 코드를 짜야한다.
- StringBuilder는 equals가 오버라이딩 되어 있지 않아서, 문자열 주소 비교가 이루어진다.
- 따라서 비교를 원할 때는, String으로 변환 후 equals로 비교한다.
