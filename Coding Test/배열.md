배열(Array)
----
- 순서(index)를 가진 데이터의 집합
- 생성과 동시에 크기가 고정된다.
- 전체 원소가 메모리상에 일렬로 저장된다.

----
- get(int idx) : idx번째 원소 반환, 시간 복잡도 O(1)

배열의 시간 복잡도 정리
----
- 원소에 접근하고 변경하는 것은 O(1)으로 빠르다.
- 단, 중간 원소를 추가/삭제하는 것은 최대 배열의 개수까지 시간이 걸린다. O(N)
