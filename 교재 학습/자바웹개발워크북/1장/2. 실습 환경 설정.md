- **JDK** : JDK 11
- **Build System** : Gradle
- **Version** : Java EE 8

맥에서 톰캣 실행 오류 시
------------------------
- macOS의 보안 문제로 외부에서 받은 실행파일(.sh)은 차단됨.
```terminal
cd 톰캣 설치 경로(bin 폴더)
chmod +x *.sh
```
- **cd** : change directory, 폴더 이동
- **chmod** : 권한 변경 명령어
- **+x** : 실행 권한 추가
- <strong>*.sh</strong>: 확장자가 .sh로 끝나는 모든 파일
