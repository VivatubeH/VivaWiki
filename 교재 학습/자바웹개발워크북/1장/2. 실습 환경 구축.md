맥에서 톰캣 실행 오류 발생 시 해결법
---------------------------

<img width="1324" height="173" alt="image" src="https://github.com/user-attachments/assets/79113650-4968-4fdd-a80d-a07c54c9979e" />

- macOS의 보안 문제로 외부에서 받은 실행 파일(.sh)에 대해서는 사용자가 직접 실행해도 되는 파일이라고 권한을 주지 않으면 실행을 차단함.
- 터미널 명령어를 통해 실행 권한을 주면 해결 가능.

```bash
cd /Users/isanghyeon/Desktop/project/apache-tomcat-9.0.113/bin

chmod +x *.sh
```
- 톰캣의 bin 폴더 ( 톰캣 로그에 뜬 주소를 복사해서 이동한다. 주소는 사용자마다 상이함. )
- cd : change directory, 폴더 이동
- chmod : 권한을 변경하겠다는 명령어
- +x : 실행(execute) 권한 추가
- *.sh : 확장자가 .sh로 끝나는 모든 파일에 적용
