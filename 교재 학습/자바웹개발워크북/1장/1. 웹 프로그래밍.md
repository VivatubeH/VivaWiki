### 웹 프로그래밍의 특징
- **기존 프로그래밍 방식** : 개발자가 모든 처리 과정을 직접 설계
- **웹 개발** : 개발자의 코드를 정해진 구조에 맞게 넣어야 함. ( 구조에 대한 이해가 필수 )

### 웹 프로젝트의 기본구조
<img width="822" height="194" alt="image" src="https://github.com/user-attachments/assets/9036362a-f5e6-46fb-a879-f03bd448aed7" />

- **브라우저(클라이언트 프로그램)** : 서버에 원하는 데이터를 요청하고 응답받음.  
- **서버(웹서버, WAS)** : 클라이언트가 요청한 데이터를 응답으로 보냄.
- **데이터베이스** : 영구적으로 데이터를 보관하고 운영하는 프로그램
